Smart Translator for Abantecart
 

What is Smart Translator?

The Smart Translator extension helps you to integrate full-page translation into Abantecart.

Smart Translator is designed to work with GTranslate, an enhanced third-party translation service built on the Google Translate platform. You need a GTranslate Enterprise account in order for this extension to work successfully. With GTranslate Enterprise, you get up to 10GB translation traffic and you can have your website translated into as many as 100+ languages! Take advantage of enhanced automatic translation and edit your translations on the spot. With this solution, you have full control of your translated content. The extension is totally Abantecart-independent, making no use of Abantecart's built-in translation feature! Translation is reflected in the page source code and therefore indexable by Google Search. Translations are hosted by GTranslate's CDN.

1. To get started, fill out the forms below.

Status: On / Off


2. Now go to Admin > System > Localization > Languages and configure the languages of your choice.

3. Sign up for a GTranslate Enterprise account.

4. Create a c-name for each language in your cPanel.* (Example: "fr" for French, "de" for Deutsch or German, etc.)

* NOTE: Don't create actual language subdomains since the translated content will be hosted by GTranslate's Content Delivery Network.

5. If you experience any issue with the language flags in the dropdown menu, add this code to your htaccess file:

# Added at Assistech's suggestion to fix problem of language flags in dropdown

<FilesMatch "\.(ttf|ttc|otf|eot|woff)$">
<IfModule mod_headers.c>
Header set Access-Control-Allow-Origin "*"
</IfModule>
</FilesMatch>

6. If you have any issue with the translation of currency in certain languages, simply edit the incorrect translation. For example: “€” must be changed to “XNUMX €”.

NOTE:

To access the translation editor, add "?language_edit=1" at the end of the URL for the page you want to edit. For example: https://ru.mydomain.com/store/?language_edit=1

If the URL already contains a question mark (?), use an ampersand (&) instead. Example: https://ru.mydomain.com/store/?currency=GBP&language_edit=1

If you have any questions about translation, please contact GTranslate directly. For installation questions, feel free to contact us.
